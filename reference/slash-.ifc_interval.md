# Divide an IFC interval by a time unit

Divide an IFC interval by a time unit

## Usage

``` r
# S3 method for class 'ifc_interval'
e1/e2
```

## Arguments

- e1:

  An `ifc_interval`.

- e2:

  A string: `"day"`, `"week"`, or `"month"`.

## Value

A numeric vector of the span expressed in the given unit.
